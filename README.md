# 考研数学学习助手 v2.0.0

一站式考研数学一学习平台，集知识点学习、智能规划、练习测试、AI助教于一体。

## 功能特性

### 1. 知识点学习
- **50+ 核心知识点**：覆盖微积分、线性代数、概率论三大科目
- **结构化内容**：每个知识点包含概念、公式、例题、要点
- **AI 详解**：一键获取 AI 生成的详细讲解
- **互动提问**：针对知识点向 AI 提问，深入理解
- **学习状态追踪**：标记学习中/已完成/已掌握

### 2. 学习规划
- **基础规划**：自动生成三阶段学习计划（基础→强化→冲刺）
- **AI 智能规划**：根据学习进度和薄弱点生成个性化计划
- **对话调整**：通过对话与 AI 调整学习计划
- **每日任务**：自动生成每日学习任务清单

### 3. 练习测试
- **专项练习**：按学科、难度、题型筛选题目
- **多种题型**：选择题、填空题、解答题
- **AI 生成题目**：实时生成新题目练习
- **错题本管理**：自动收录错题，支持重做和导出

### 4. AI 助教
- **多轮对话**：保持上下文的连续对话
- **数学公式输入**：15个常用公式符号快捷输入
- **公式渲染**：LaTeX 公式实时预览
- **对话历史**：保存和搜索历史对话
- **导出功能**：导出对话为 Markdown 或文本文件

### 5. 学习统计
- **多维数据**：总题数、正确率、学习天数统计
- **可视化图表**：
  - 知识点掌握雷达图
  - 正确率趋势折线图
  - 错题分布饼图
  - 每日练习量柱状图
  - 学习热力图日历
- **学习报告**：生成 HTML 格式学习报告

### 6. 设置管理
- **学习偏好**：每日学习时长、目标分数、薄弱科目
- **存储空间**：可视化显示数据使用情况
- **分类导出**：单独导出学习记录/错题本/AI对话/学习规划
- **分类清空**：单独清空各类数据，二次确认防误操作

## 快速开始

### 1. 下载项目

```bash
git clone https://github.com/your-username/math.git
cd math
```

### 2. 直接打开

双击 `考研数学学习助手.html` 在浏览器中打开即可使用。

> 推荐使用 Chrome、Edge 或 Firefox 浏览器

### 3. 配置 AI

首次使用需要配置 AI 模型：

1. 点击左侧导航栏的「设置」
2. 在 AI 配置区域选择厂商（支持 Claude、Deepseek、智谱AI、OpenAI兼容）
3. 输入 API Key
4. 点击「测试连接」验证配置
5. 保存配置

## 技术栈

- **前端框架**：原生 JavaScript (ES6+)
- **样式**：CSS3 + Flexbox
- **数学公式**：MathJax 3
- **图表库**：Chart.js
- **热力图**：Cal-Heatmap
- **数据存储**：localStorage
- **AI 接口**：Claude API / Deepseek API / 智谱 API / OpenAI 兼容 API

## 支持的 AI 模型

| 厂商 | 模型 |
|------|------|
| Claude (Anthropic) | claude-3-5-sonnet, claude-3-opus, claude-3-sonnet |
| Deepseek | deepseek-chat, deepseek-coder |
| 智谱AI | glm-4, glm-3-turbo |
| OpenAI 兼容 | 自定义模型 |

## 数据说明

所有数据存储在浏览器的 localStorage 中，包括：

- `mathHelper_progress` - 学习进度
- `mathHelper_learningProgress` - 知识点学习状态
- `mathHelper_learningNotes` - 学习笔记
- `mathHelper_wrongQuestions` - 错题本
- `mathHelper_studyPlans` - 学习规划
- `mathHelper_aiConversations` - AI 对话记录
- `mathHelper_aiConfig` - AI 配置
- `mathHelper_settings` - 用户设置

> 建议定期导出数据备份，防止数据丢失

## 使用建议

1. **制定计划**：首先在「学习规划」模块生成学习计划
2. **系统学习**：按照计划在「知识点」模块学习各知识点
3. **及时练习**：学完知识点后在「练习」模块做题巩固
4. **错题复习**：定期复习错题本中的题目
5. **AI 答疑**：遇到问题随时使用「AI 助教」提问
6. **定期统计**：在「统计」模块查看学习情况，调整策略
